# Crezcamos Juntos - Terapia Pediátrica

Landing page para el centro de rehabilitación pediátrica **Crezcamos Juntos**.

## Características
- [x] Diseño premium con **Tailwind CSS**.
- [x] Tipografía moderna (**Plus Jakarta Sans** & **Manrope**).
- [x] Iconografía de **Material Symbols**.
- [x] Efectos de **Glassmorphism** y animaciones fluidas.
- [x] Adaptable a dispositivos móviles.

## Desarrollo
Para visualizar el proyecto localmente, puedes abrir el archivo `index.html` directamente en tu navegador o usar un servidor estático:

```bash
npx serve .
```

## Estructura
- `index.html`: Página principal con todos los estilos y estructura.
